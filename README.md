---

## Project Description

This project was developed for IT3040 ITPM Assignment 1. The purpose of this project is to test the accuracy of the Chat Sinhala transliteration function available on the Pixelssuite Chat Translator website.

The website converts chat-style Singlish input into Sinhala output. This project focuses on identifying cases where the system fails to correctly convert informal Singlish words, phrases, and sentences into the expected Sinhala output.

The testing process is automated using Python and Playwright. Test cases are stored in an Excel file. The automation script reads each Singlish input from the Excel sheet, enters it into the website, captures the actual Sinhala output generated by the website, compares it with the expected Sinhala output, and records the result in the Excel file.

## Command Explanation

| Command Part | Meaning |
|---|---|
| python test_automation.py | Runs the automation script |
| --excel "IT23654662_Assignment 1 - Test cases.xlsx" | Specifies the Excel file containing test cases |
| --url "https://www.pixelssuite.com/chat-translator" | Specifies the website to test |
| --wait-ms 15000 | Waits for the website output to load |
| --retry-wait-ms 2000 | Waits between retry attempts |
| --retries 10 | Retries output reading multiple times |
| --type-delay-ms 80 | Adds delay while typing input |
| --slow-mo-ms 300 | Slows down browser actions for visibility |
| --save-every 1 | Saves the Excel file after each test case |
| --keep-open | Keeps the browser open after execution |

## Result Interpretation

| Status | Meaning |
|---|---|
| Pass | Actual output matched the expected output |
| Fail | Actual output did not match the expected output |

Since this assignment focuses on failures in the transliteration system, the main expected result is Fail for the identified negative test cases.

---

## Important Notes

This project uses strict comparison between expected output and actual output.

A test case may fail because of small differences such as:

- Sinhala spelling differences
- Incorrect transliteration
- Missing words
- Extra words
- Incorrect spacing
- Punctuation differences
- Formatting differences
- Incorrect handling of English words
- Incorrect handling of numbers, dates, times, currency, links, or emojis

These failures are useful because the main objective is to identify weaknesses in the Chat Sinhala transliteration system.

## Common Issues and Fixes

### 1. Playwright command is not recognized

Use:

python -m playwright install

### 2. Excel permission denied error

Close the Excel file before running the automation script.

### 3. Excel file not found error

Make sure the Excel file is in the same folder as test_automation.py.

Example:

python IT23654662_test_automation.py --excel "C:\Users\User\Desktop\IT23654662\IT23654662_Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator"

### 4. Website loading slowly

Increase the wait time:

--wait-ms 20000

### 5. Some negative test cases show Pass

Replace those inputs with more complex Singlish inputs and run the automation again.
